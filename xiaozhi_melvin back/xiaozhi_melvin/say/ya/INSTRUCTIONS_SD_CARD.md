# Инструкция по подготовке SD карты для робота Мелвина

## Шаг 1: Проверьте созданные WAV файлы
Файлы уже созданы в директории `python416/wav_files/`:
```
wav_files/
├── hello.wav                    (тестовый тон 1 кГц)
├── test_tone_440.wav           (тон 440 Гц)
├── chirp.wav                   (чирп-сигнал)
├── beeps.wav                   (3 бипа)
├── voice_like.wav              (голосоподобный)
├── привет_я_мелвин.wav         (TTS фраза 1)
├── готов_к_работе.wav          (TTS фраза 2)
├── батарея_заряжена.wav        (TTS фраза 3)
├── подключение_установлено.wav (TTS фраза 4)
└── ошибка_проверьте_соединение.wav (TTS фраза 5)
```

## Шаг 2: Подготовьте SD карту
1. Вставьте SD карту в компьютер
2. Отформатируйте её в FAT32 (если ещё не отформатирована)
3. Запомните букву диска (например, `D:`)

## Шаг 3: Скопируйте файлы вручную

### Вариант A: Через Проводник Windows
1. Откройте директорию `python416/wav_files/`
2. Выделите все файлы (Ctrl+A)
3. Скопируйте (Ctrl+C)
4. Откройте SD карту в Проводнике
5. Вставьте файлы (Ctrl+V)

### Вариант B: Через командную строку
```cmd
# Откройте командную строку от имени администратора
cd python416
xcopy /Y wav_files\*.wav D:\
mkdir D:\tts
xcopy /Y wav_files\*.wav D:\tts\
```
Замените `D:` на букву вашей SD карты.

### Вариант C: Через PowerShell
```powershell
cd python416
Copy-Item -Path "wav_files\*.wav" -Destination "D:\" -Force
New-Item -ItemType Directory -Path "D:\tts" -Force
Copy-Item -Path "wav_files\*.wav" -Destination "D:\tts\" -Force
```
Замените `D:` на букву вашей SD карты.

## Шаг 4: Проверьте содержимое SD карты
После копирования на SD карте должно быть:
- Все WAV файлы в корне
- Директория `tts` с теми же файлами

## Шаг 5: Вставьте SD карту в плату
1. Извлеките SD карту из компьютера
2. Вставьте в слот на плате SpotPear ESP32-S3
3. Убедитесь, что карта плотно вставлена

## Что дальше?

### 1. Дождитесь завершения компиляции
PlatformIO компилирует проект. Когда увидите `[SUCCESS]`, переходите к следующему шагу.

### 2. Загрузите прошивку
```bash
cd python416
platformio run --target upload
```

### 3. Откройте Serial Monitor
```bash
platformio device monitor --baud 115200
```

### 4. Протестируйте систему
- **Одиночное нажатие BOOT**: `hello.wav`
- **Двойное нажатие**: случайная TTS фраза
- **Тройное нажатие**: все TTS фразы по порядку

## Устранение проблем

### Если нет звука:
1. Проверьте подключение динамика/наушников
2. Убедитесь, что усилитель включен (GPIO46)
3. Проверьте формат WAV файлов (16-bit, 16kHz, mono)

### Если файлы не читаются:
1. Переформатируйте SD карту в FAT32
2. Убедитесь, что файлы скопированы в корень
3. Проверьте, что карта плотно вставлена

### Если TTS не работает:
1. Убедитесь, что файлы есть в директории `/tts/`
2. Проверьте, что SD карта инициализирована (см. Serial Monitor)

## Готово!
После выполнения этих шагов робот Мелвин сможет воспроизводить звуки и "говорить" тестовые фразы.
