# SpotPear ESP32-S3-1.54 V2.0 - Воспроизведение WAV с SD карты

Проект для воспроизведения аудиофайлов с SD карты на плате SpotPear ESP32-S3-1.54 V2.0 с использованием Arduino framework через PlatformIO.

## 📋 Требования

### Аппаратные
- Плата SpotPear ESP32-S3-1.54 V2.0
- Micro-SD карта (формат FAT32)
- Динамик или наушники, подключенные к аудиоразъему платы
- Кабель USB-C для питания и программирования

### Программные
- PlatformIO IDE (VS Code с расширением PlatformIO)
- Драйверы USB для ESP32-S3
- Аудиофайл `hello.wav` на SD карте

## 🎛️ Распиновка платы

| Компонент | GPIO Пины |
|-----------|-----------|
| **I2C (ES8311)** | SDA=15, SCL=14 |
| **I2S (ES8311)** | MCLK=16, BCLK=9, LRCK=45, DOUT=10, DIN=11 |
| **Усилитель (NS4150B)** | PA_CTRL=46 |
| **SD Карта** | CLK=38, CMD=40, D0=39 |
| **Кнопка BOOT** | GPIO0 |

## 📁 Подготовка SD карты

1. Отформатируйте SD карту в FAT32
2. Создайте аудиофайл `hello.wav` со следующими параметрами:
   - Формат: WAV (PCM)
   - Частота дискретизации: 16000 Гц или 44100 Гц
   - Битность: 16 бит
   - Каналы: Моно или Стерео
3. Скопируйте файл `hello.wav` в корень SD карты
4. Вставьте SD карту в слот на плате

```ini

```

## 🚀 Загрузка прошивки

1. Подключите плату через USB-C
2. В PlatformIO нажмите "Upload" (стрелка вправо)
3. Дождитесь завершения загрузки

## 🔊 Использование

1. После загрузки прошивки откройте Serial Monitor (115200 baud)
2. Вы увидите процесс инициализации:
   - Включение усилителя NS4150B
   - Инициализация SD карты
   - Поиск файла `hello.wav`
   - Начало воспроизведения

3. Для повторного воспроизведения нажмите кнопку BOOT на плате

## 🔍 Диагностика проблем

### SD карта не определяется
- Проверьте формат карты (должен быть FAT32)
- Убедитесь, что карта вставлена до включения питания
- Проверьте соединение пинов SD карты

### Нет звука
- Проверьте подключение динамика/наушников
- Убедитесь, что усилитель включен (GPIO46 = HIGH)
- Проверьте формат аудиофайла
- Увеличьте громкость в коде: `audio.setVolume(15)`

### Ошибки компиляции
- Убедитесь, что установлены все библиотеки:
  - `arduino-libraries/SD`
  - `earlephilhower/ESP32-audioI2S`
- Проверьте версию PlatformIO (рекомендуется 6.6.0)

## 📚 Используемые библиотеки

## 🔄 Возврат к ESP-IDF

Если нужно вернуться к ESP-IDF версии:
1. Переименуйте `main.cpp` обратно в `main.c`
2. Восстановите оригинальный `platformio.ini`
3. Используйте `main_idf.c` как основной файл
