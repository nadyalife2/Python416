# Робот Мелвин - Система TTS (Text-to-Speech)

## Обзор
Реализация системы синтеза речи для робота Мелвина на базе ESP32-S3 (SpotPear ESP32-S3-1.54 V2.0). Система позволяет роботу генерировать речь из текста в реальном времени.

## Особенности
- Воспроизведение WAV файлов с SD карты
- Синтез речи из текста (TTS) через библиотеку ESP32-TTS
- Управление через кнопку BOOT (одинарное/двойное/тройное нажатие)
- Поддержка русского языка
- Кэширование сгенерированной речи на SD карту

## Требования к оборудованию

### Аппаратные
- Плата SpotPear ESP32-S3-1.54 V2.0
- Micro-SD карта (формат FAT32, минимум 1 ГБ)
- Динамик 8Ω 1W или наушники
- Кабель USB-C для питания и программирования

### Программные
- PlatformIO IDE (VS Code с расширением PlatformIO)
- Python 3.x (для создания тестовых WAV файлов)
- Драйверы USB для ESP32-S3

## Быстрый старт

### 1. Подготовка SD карты
```bash
# Перейдите в директорию проекта
cd python416

# Создайте тестовые WAV файлы
python create_melvin_wavs.py

# Скопируйте файлы из wav_files/ в корень SD карты
# Рекомендуемые файлы для копирования:
# - hello.wav
# - voice_like.wav
# - Все файлы из wav_files/tts/ (если созданы)
```

### 2. Настройка проекта
```bash
# Установите зависимости PlatformIO
cd python416
platformio lib install
```

### 3. Компиляция и загрузка
```bash
# Компиляция проекта
platformio run

# Загрузка на устройство (порт COM5 по умолчанию)
platformio run --target upload

# Если порт другой, измените upload_port в platformio.ini
```

### 4. Мониторинг Serial порта
```bash
# Откройте Serial Monitor
platformio device monitor --baud 115200
```

## Использование

### Запуск системы
1. Подключите плату через USB-C
2. Вставьте SD карту с тестовыми WAV файлами
3. Загрузите прошивку
4. Откройте Serial Monitor (115200 бод)

### Управление кнопкой BOOT
- **Одиночное нажатие**: Воспроизведение hello.wav
- **Двойное нажатие**: Случайная тестовая фраза TTS
- **Тройное нажатие**: Все тестовые фразы TTS по порядку

### Тестовые фразы TTS
Система включает 5 тестовых фраз на русском:
1. "Привет, я Мелвин"
2. "Готов к работе"
3. "Батарея заряжена"
4. "Подключение установлено"
5. "Ошибка, проверьте соединение"

## Архитектура TTS системы

### Компоненты
1. **MelvinTTS класс** - высокоуровневый API для синтеза речи
2. **ESP32-TTS библиотека** - низкоуровневый синтезатор речи
3. **Аудио подсистема** - I2S драйвер + кодек ES8311
4. **SD карта** - хранение WAV файлов и кэша TTS

### Поток данных
```
Текст → MelvinTTS → ESP32-TTS → PCM данные → I2S драйвер → ES8311 кодек → Усилитель → Динамик
```

## Расширенные возможности

### Добавление новых фраз
```cpp
// В файле MelvinTTS.cpp добавьте новую фразу в массив TEST_PHRASES
const char* MelvinTTS::TEST_PHRASES[6] = {
    "Привет, я Мелвин",
    "Готов к работе", 
    "Батарея заряжена",
    "Подключение установлено",
    "Ошибка, проверьте соединение",
    "Новая фраза"  // ← Добавьте здесь
};
```

### Использование TTS API в коде
```cpp
// Инициализация
MelvinTTS tts;
tts.begin();

// Синтез речи
tts.speak("Привет, мир!");

// Случайная фраза
tts.speakRandomPhrase();

// Конкретная фраза по индексу
tts.speakTestPhrase(0);  // "Привет, я Мелвин"

// Управление громкостью (0-21)
tts.setVolume(15);
```

### Кэширование речи на SD карту
Система автоматически проверяет наличие предварительно сгенерированных WAV файлов в директории `/tts/` на SD карте. Если файл существует, он воспроизводится напрямую, что ускоряет работу и снижает нагрузку на процессор.

## Устранение неполадок

### Проблема: Нет звука
**Возможные причины:**
1. Неправильная распиновка
2. Усилитель не включен (GPIO46)
3. Неправильный формат WAV файла
4. SD карта не отформатирована в FAT32

**Решение:**
1. Проверьте вывод в Serial Monitor
2. Убедитесь, что функция `enableAmplifier()` вызывается
3. Проверьте формат WAV файла (16-bit, 16kHz, mono)
4. Отформатируйте SD карту в FAT32

### Проблема: Шипение/шум
**Возможные причины:**
1. Проблемы с заземлением
2. Помехи от WiFi
3. Некачественный усилитель

**Решение:**
1. Добавьте фильтрующий конденсатор 100µF на выходе усилителя
2. Отключите WiFi во время воспроизведения
3. Используйте экранированный аудиокабель

### Проблема: TTS не работает
**Возможные причины:**
1. Библиотека ESP32-TTS не установлена
2. Недостаточно памяти
3. SD карта не доступна для кэширования

**Решение:**
1. Проверьте установку библиотеки в platformio.ini
2. Включите PSRAM в настройках проекта
3. Убедитесь, что SD карта правильно инициализирована

## Оптимизация производительности

### Использование PSRAM
Включите PSRAM в `platformio.ini`:
```ini
build_flags =
    -DBOARD_HAS_PSRAM
    -mfix-esp32-psram-cache-issue
```

### Уменьшение частоты дискретизации
Для экономии памяти можно уменьшить частоту дискретизации с 16 кГц до 8 кГц в коде TTS.

### Предварительная генерация фраз
Сгенерируйте часто используемые фразы заранее и сохраните их как WAV файлы на SD карте.
